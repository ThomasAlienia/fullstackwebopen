# React + Vite

How to run locally: 
- Open Integrated Terminal at folder part2.12_2.15
- Run  `npm install`
- Run  `npm install axios`
- Run  `npm install json-server --save-dev`
- Run `npm run server`
- Open/split another Integrated Terminal at folder part2.12_2.15
- Run `npm run dev`
